A responsive e-commerce app built with React, Redux Toolkit, and Tailwind CSS.

## Features

- Browse products from Fake Store API

-Add/remove items in cart with real-time updates

-Cart summary: total items & amount

-Responsive: mobile, tablet, desktop

-Toast notifications for actions

-Loading spinner while fetching products

## Tech Stack

React, React Router, Redux Toolkit, Tailwind CSS, React Hot Toast, React Icons

## Installation
- Install dependencies
npm install

-Start development server
npm start


Open http://localhost:3000 in browser, if not working run in another consecutive port

## Usage

-Browse products on Home page

-Click Add to Cart to add items

-Click Cart icon to view cart

-Remove items with delete icon

## State Management

-Managed with Redux Toolkit

-Actions: add(product) | remove(productId)

## API

-Endpoint: https://fakestoreapi.com/products

-Returns products with id, title, price, description, image, etc.

## Styling

-Tailwind CSS with Navy Blue & Gold theme

-Fully responsive with hover effects
